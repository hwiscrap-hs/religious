## 1. 종교의 사회적 기능 (4가지)

> 기능의 우열을 비교하는 것이 아니라 인간·문화 이해의 렌즈로 사용

### (1) 정당화 기능 (Justification)
사회의 규범(Norm)과 구조(Structure)를 종교적으로 정당화하여 구성원들이 이를 수용하게 함

| 사례 | 정당화 내용 |
|------|-----------|
| 기독교의 매장 문화 | 부활을 위해 시신 보존 필요 |
| 티베트 조장(天葬) | 불교 윤회관에 따른 수용 |
| 인도 카스트 제도 | 힌두교적 정당화로 헌법 폐지(1947년) 후에도 유지 |
| 호주제·동성동본금혼 | 유교적 정당화로 유지 → 정당화 약화 후 폐지 |
| 봉건 신분제 | 신이 황제·교황을 임명했다는 종교적 정당화 |

- 종교적 정당화가 약해지면 사회 제도도 변화 (동성결혼 합법화 등)

### (2) 정체성/수행적 기능 (Performative Function)
공동 의례 참여를 통해 "우리는 같은 공동체"라는 집단 정체성 형성

| 종교 | 공동체 용어 | 특징 |
|------|-----------|------|
| 기독교 | 교회(Ekklesia) | "불러 모으다" = 사람들의 모임 |
| 불교 | 승가(Sangha) | 비구·비구니·우바새·우바이 4부대 |
| 이슬람 | 움마(Ummah) | 모든 무슬림의 거대 공동체 |

### (3) 자기주도적 기능 (Holistic Function)
개인이 자신의 목표를 스스로 찾고 달성할 수 있도록 돕는 역할
- 개인 욕망의 종교적 해소 → 사회 불만 최소화 → 사회 통합
- 사례: 동남아 사원의 제물 문화 (소망 성취 후 감사 제물)

### (4) 유화적 기능 (Literative Function)
사회 갈등과 분쟁을 부드럽게 해소하고 중재하는 역할
- **기독교 고해성사**: 신부에게 죄를 고백 → 심리적 불안 해소 (신부 비밀유지 의무)
- **구약 도피성**: 실수로 살인한 자를 제사장이 중재하여 복수의 악순환 방지

> ⚠️ 역기능도 있음: 종교가 사회 분열·파괴를 초래하기도 함

## 2. 통과의례(Rite of Passage)와 정체성

> Arnold van Gennep 이론 | 삶의 각 단계를 넘어가며 정체성을 재확인하는 종교 의식

### 인간의 주요 발달 단계
1. **5~7세**: 제1차 성징 인식 (남녀 구별 시작)
2. **13~14세 사춘기**: 제2차 성징 (급격한 신체 변화 → 많은 문화에서 성인식 실행)
3. **결혼**: 호르몬 변화 (남성은 차분해지고 여성은 활달해짐)
4. **40대 갱년기**: 육체적 능력 저하 (사회적 책임 증대)
5. **60~70대 노년기**: 성적 기능 상실, 새로운 정체성 형성

### 통과의례의 3단계

| 단계 | 내용 |
|------|------|
| **1단계: 분리(Separation)** | 이전 지위에서 분리 (격리, 특별 표식 부여) |
| **2단계: 한계/이행(Liminal)** | 상징적 죽음 경험, 극심한 육체·정신적 고통 (벌집에 쏘이기, 나무에서 뛰어내리기 등) |
| **3단계: 재결합(Aggregation)** | 새로운 지위로 사회 복귀 (할례, 새 이름·옷 부여) |

### 종교의 정체성 기능 (2가지)
1. **육체와 정신의 조화**: 신체 변화를 정신적으로 인지하고 수용하도록 도움
2. **집단 정체성 확립**: 공동체 소속을 통해 온전한 사회인이 됨

- **실생활 응용**: 교도소 입소식·퇴소식도 통과의례의 원리 → 재범률 감소 효과

## 3. 죽음과 종교

### 종교에서 보는 죽음의 특성
- 죽음 확률: **100%** (누구나 반드시 죽음)
- **유일성**: 딱 한 번만 경험, 아무도 대신 불가
- **절대성**: 모든 인간의 필수 경험

### 종교별 죽음 이후 관점

| 종교 | 관점 | 의미 |
|------|------|------|
| 불교·도교 | **환생(輪生)** | 죽음 = 환생의 시작점 (죽지 않으면 환생 불가) |
| 기독교·이슬람 | **부활(復活)** | 신체적/유전자적/영적 부활 (믿음의 영역) |
| 이집트 문명 | **영혼불멸설** | 미라·피라미드 → 육체 보존 후 영혼 재입 |

> 모든 종교 공통: 인간은 육체 외에 다른 요소(영혼/혼/기(氣)) 존재를 상정

### 쿨러-로스(Kübler-Ross)의 죽음 수용 5단계
저서: 『죽음과 죽어감』(On Death and Dying)

1. **부정(Denial)**: 진단 사실 거부, 다른 의사 찾아다님
2. **분노(Anger)**: "왜 나인가?" 원망
3. **흥정(Bargaining)**: 신과의 거래 시도, 죽음 미루기 시도
4. **우울(Depression)**: 타협 불가능 인식, 깊은 우울
5. **수용(Acceptance)**: 죽음 수용, 화해와 용서, 의미 있는 삶 추구

### 장례식의 근본적 의미
> **"살아있는 사람들의, 살아있는 사람들에 의한, 살아있는 사람들을 위한 의식"**

| 단계 | 대상 | 의미 |
|------|------|------|
| 임종(臨終) | 가족·친척 | 살아있는 사람이 할 수 있는 마지막 효도 |
| 문상(問喪) | 살아있는 유족 | 죽은 자가 아닌 살아있는 자를 위로 |
| 제사(祭祀) | 가족·친척 | 망자 추모 → 살아있는 자의 위로 |

- 죽음에 대한 성찰이 많은 사람이 더 의미 있는 삶을 삶
- 인간의 근본 능력: **Hope Ability (희망할 수 있는 능력)**

## 4. 일상적 삶의 의미와 종교

### 핵심 개념
- **의미 부여(Meaning-Making)**: 인간은 의미를 먹고 사는 존재
- 의미가 있으면 어떤 고통도 견딜 수 있고, 죽음도 삶의 의미로 전환 가능

### 의미 부여의 사례
- 어린아이의 질문: "저 꽃은 왜 예뻐요?" → 과학적 설명보다 의미 있는 이야기에 만족
- 이름 짓기: 한자 하나하나에 의미를 담아 자녀의 정체성 형성
- 달무리·까치 울음: 과학적 근거 없지만 하루를 희망적으로 시작하게 하는 심리적 효과
- 유대교 할레 의식: "하나님과의 약속"이라는 의미 부여로 고통스러운 의식 수용

### 죽음학(Thanatology)의 역설
- 죽음학은 죽음 자체가 아니라 **삶의 의미**를 탐구하는 학문
- 동양의 오복(五福) 중 **고종명(考終命)**: 주어진 명을 다 누리고 죽는 것 = 가장 큰 복

### 종교의 의미 부여 역할
- 고통과 죽음에 의미를 부여하여 인간이 더 의미 있게 살도록 도움
- 자연현상(달무리, 까치 등)에도 문화적 의미 부여
- 연인 이별을 "인연이 없어서"로 표현 → 외적 힘(운명)에 귀인하여 심리적 안정

## 5. 놀이와 종교의례의 공통성

### 호모 루덴스(Homo Ludens) - 요한 호이징아
> 인간은 이성적 존재(호모 사피엔스)일 뿐만 아니라 **놀이하는 존재**

- **놀이의 특성**: 목적이 없음, 결과를 바라지 않음, 활동 자체가 목적, 순수한 내적 욕구
- 현대 최대 문제: 일(Work)만 중요시하고 놀이(Play)를 시간낭비로 봄

### 놀이와 종교의례의 4가지 공통점

| 공통점 | 놀이 | 종교 의례 |
|--------|------|----------|
| **특정 시간·공간 필요** | 운동장, 오후 | 일요일(기독교), 금요일(이슬람), 성소·사원 |
| **역할 참여** | 각자 역할 수행 | 모든 신자가 능동적으로 참여 |
| **새로운 세계 창조** | 고유 법칙, 기존 지위 무시 | 종헌(종교단체 헌법), 정교분리 |
| **안전한 표현의 장** | 결과 염려 없이 자유 활동 | 심판 없이 자유롭게 표현 |

### 종교의 표현적 기능 (Expressive Function)
- 일상에서 표현하지 못하는 것들을 자유롭게 표현할 기회 제공
- **고해성사**: 비밀 보장 → 안전한 표현 → 심리적 정화
- **주술**: 목적 달성이 아니라 억울함·분노를 표현하는 수단 (현대적 재해석)
- 킹 미다스 이야기: 비밀을 혼자 안고 있으면 미쳐감 → 표현의 필요성

### 올림픽과 종교 의례
- 올림픽 어원: "올림픽 제전(Festival)" = 종교 의식에서 유래
- 개막식·폐막식이 경기보다 중요 (참여 자체가 의미)
- 원래 아마추어 정신 → 현대 상업화로 변질

## 6. 세계 종교의 지역별 분포

### 기독교
- **천주교 중심**: 중남미, 유럽 남부, 아프리카 중부

### 불교 (3가지 분파)
| 분파 | 지역 |
|------|------|
| **대승불교** | 한국, 중국, 일본 |
| **상좌부불교(소승)** | 미얀마, 스리랑카, 라오스, 인도 남부 |
| **티베트불교** | 티베트 지역, 부탄(국교) |

### 이슬람교
- 중동 대부분, 아프리카 북부
- 아시아: 인도네시아, 말레이시아, 파키스탄, 방글라데시

### 힌두교
- 인도 인구의 80% 이상 (초등학교 교과서부터 등장)
- 현재 인도의 국제적 중요성 상승

## 🔑 추가 핵심 개념 (배경 지식)

### 종교의 정의
- **한자 어원**: 宗(으뜸/제사) + 敎(가르침) = "모든 가르침 중 으뜸이 되는 최고의 가르침"
- **영어 어원**: Religion = Re(다시) + Ligare(연결) = 신과 인간의 관계 회복
- **작업가설적 정의(Operational Definition)**: 완벽한 정의는 불가능하므로 수정·보완 가능한 임시 정의 사용
- **3가지 핵심 개념**: 성(Sacred, 거룩함) · 초월(Transcendence) · 궁극(Ultimate)

### 종교 연구의 윤리 (3원칙)
1. **존중의 원칙**: 복장·금지사항 확인, 타 문화 신성함 훼손 금지
2. **중립성의 원칙**: "이단", "사이비" 판단 금지
3. **개인 신앙 분리 원칙**: 종교 공부 ≠ 종교 신앙 (신앙 변화 시 공부가 제대로 안 된 것)

### 신화와 상징
- **신화**: 의도적으로 체계화된 의미 전달 체계 (음악의 화성법처럼)
- **상징(Symbol)**: 어원 Sym(함께) + Ballein(던지다) = 의미를 전달하기 위한 도구
- **상징 해석 3가지 방법론**:
  - **구조주의**: 개별 상징이 구조적으로 연결되어 의미 체계 형성
  - **기능주의**: 상징이 사회적 목적을 위해 의도적으로 창조됨
  - **해석학적 접근**: 맥락(Context) 속에서 의미 추구 (종교학이 강조)

### 주요 용어 구분
| 용어 | 정확한 의미 |
|------|-----------|
| **미신(迷信)** | 사람들을 미혹하여 이용하려는 신앙 (비과학적이라고 무의미한 것은 아님) |
| **이단(異端)** | 같은 종교 내에서 정통이 다른 주장을 지칭하는 것 (종교 간 구분 아님) |
| **사이비(似而非)** | 종교와 비슷하지만 종교가 아닌 것 ("사이비 종교"는 중복 표현) |

## 📝 답안 작성 요령

1. **개념 정의** → **구체적 사례** 순서로 서술
2. 사례: 인도 축제, 티베트 순례(삼보일배), 카스트 제도, 교도소 입소·퇴소식, 고해성사, 올림픽 등
3. 주제 간 연결 문제 가능 → 예: "통과의례와 정체성 기능의 관계", "놀이와 종교의례의 공통성이 표현적 기능과 어떻게 연결되는가"
4. **핵심 키워드**: 신화와 상징의 이해, 종교 = 인간·문화 이해
